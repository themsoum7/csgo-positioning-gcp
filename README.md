# csgo-positioning-gcp

So far it only works with old de_dust 2 map.

## How to run?
1. Go into `server` folder 
2. Run cmd  
3. Run these commands 
   * `set FLASK_APP=server.py`
   * `flask run`
4. Go to http://127.0.0.1:5000
5. You're done
